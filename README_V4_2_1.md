# JKR Investments v4.2.1 — Light Theme Contrast Patch

This patch changes only `assets/css/styles.css`.

It improves the light theme by:
- replacing grey-on-grey cards with white and pale blue-grey surfaces;
- darkening headings, labels, subtitles, and card text;
- strengthening borders and table contrast;
- improving action-centre, sector, holding, technical, and result-card readability;
- slightly increasing small supporting text sizes.

No Supabase, portfolio, market-data, JavaScript, or broker files are changed.
